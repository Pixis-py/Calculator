# DevOps TD1 : Calculator
Ce projet a pour but d'apprendre la bonne utilisation de github dans le cadre du module devops,
permettant une bonne gestion de projet a dépot partagé.
Pour installer les dépendances créer un espace virtuel avec python -m venv venv activer votre environnement.
Une fois activer installer les dépendances via pip install -r requirements.txt..
Pour éxecuter les tests utiliser cette commande python -m coverage run -m unittest tests/test_calculator.py, et voir le rapport de test avec coverage coverage report -m.

## Fonctionnalités
- Addition
- Multiplication
